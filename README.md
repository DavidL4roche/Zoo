# Description de l'archive:
L'archive est séparée en différents dossiers :
- La spécification minimale (documents demandés pour le rendu) sont dans le dossier documents
- Le code est à la racine et vous trouverez le code principal dans le dossier src (c'est un projet Eclipse)
- L'application au format JAR est dans le dossier builts
- Le manuel d'utilisation est à la racine : c'est le fichier manuel.md

# Informations
Concernant la Javadoc, il nous a été impossible de la mettre en place du à un problème d'encodage (UTF-8).
Nous nous en excusons, mais le code étant correctement indenté et commenté, vous ne devriez pas avoir de mal à le comprendre.

Lorsque vous exécuterez le fichier Zoo.jar, seul le scénario des loups vous sera montré car il est le plus pertinent dans notre projet.
Si vous souhaitez voir comment fonctionne le Zoo en général (animaux, employé, enclos), il vous suffit de décommenter la première partie du fichier ZooMain.java (comme précisé dans le code)
De plus, si vous exécuterez le fichier Zoo.jar, les caractères spéciaux (alphabet grec) ne sont pas reconnus et sont remplacés par des '?'.
